et-otp by ecki

et-OTP - eckes' TOTP Java Soft Token

Compatibility

The software is tested to work with Amazon MFA for AWS, Dropbox Two-step verification, Google 2-step verification as well as Microsoft Two-Step verification. In all cases the text string code (instead of the QR-Code) needs to be copied and pasted to the et-otp settings dialog to enable this use case. See the following GitHub Wiki pages for a more detailed description:

  • Setting up Google 2-step verification
  • Setting up Amazon MFA
  • Setting up Microsf 2-Step verification
  • Setting up Dropbox MFA

Dependencies

Java SE (Swing), JCE. Tested with Oracle Java SE version 6 and 7.

Install

Copy the et-otp.jar to a writeable directory and execute it as executable JAR. 

java -jar et-otp.jar
It will generate a .et-otp.properties file when the settings have been saved.
